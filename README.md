## Real-Time-Chat-Application

## HOW TO RUN  WEBSITE
## FRONTEND:- Write on terminal 
          (1) cd client
          (2) npm install
          (3) npm run dev


## BACKEND:- Write on terminal 
          (1) Set Up the Backend (Node.js + Express + Socket.io )
          (2) Install Node.js (if not already installed).

## Run the App:- (2) Start backend server:- node server.js
               or, npm run server
